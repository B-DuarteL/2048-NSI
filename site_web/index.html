Jeu 2048

ce site présente notre projet de nsi qui est le jeu 2048 réalisé en python.

Régle du jeu

  • Le jeu se joue sur une grille 4 x 4 cases.
  • Le joueur utilise les fléches du clavier pour déplacer les tuiles.
  • Deux tuiles de même valeur fusionnent lorsqu'elles se rencontrent.
  • Aprés chaque mouvement valide, une nouvelle tuile apparaît.
  • Le joueur gagne lorsqu'il atteint la tuile 2048.
  • La partie est perdu lorsque plus aucun mouvement n'est possible.

Fonctionnement algorithme

Le plateau de jeu est représenté par une grille 4x4 en Python. Chaque case contient soit 0 (case vide) ou une puissance de 2

Lorsqu'un mouvement est effectué, le programme marche en trois étapes :

  1. Tassement : Les tuiles se déplace vers la direction choisie.
  2. La fusion : Deux tuiles identique fusionnent en une seule.
  3. Nouvelle modification pour supprimer les cases vides créés.

Les mouvement vers le haut, le bas, la gauche et la doite ont obtenus grâce à la transformation de la grille(inversion et transportation)

Présentation du programme

le jeu à été programmé en Python en utilisant les bibliothéque Tkinter pour l'interface graphique et Random pour génerer des nombres aléatoires.
